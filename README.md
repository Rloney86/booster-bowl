Booster Bowl – Live Deployment
# Booster Bowl (Next.js Starter)

This project is ready to deploy on **Vercel** via **GitHub**.

## Run locally
```bash
npm install
npm run dev
# open http://localhost:3000
```

## Deploy with Vercel
1. Push these files to a GitHub repo.
2. Go to https://vercel.com → Add New → Project → Import your repo.
3. Click Deploy (Next.js will be auto-detected).

## Notes
- Folder structure:
  - /pages — contains route files (required by Next.js)
  - /styles.css — simple global style
  - next.config.js — Next.js config
  - package.json — dependencies
